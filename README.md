## NestJS Blog REST API (+Authentication, Swagger)

Prisma:

```
npm i prisma -D
npm i @prisma/client
npx prisma init
npx prisma migrate dev
npx prisma migrate dev --name init
```

Authentication:

```
npm i @nestjs/passport @nestjs/jwt passport-jwt
npm i @types/passport-jwt -D
npm i bcrypt
npm i @types/bcrypt -D
```
